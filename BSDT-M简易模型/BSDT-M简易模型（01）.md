1）输入区

| 列 | 字段       | 含义    | 填写                  |

| - | -------- | ----- | ------------------- |

| A | 日期       | 分析日期  | 手动                  |

| B | 中位价      | 当前价格  | 手动/导入               |

| C | 上方区间顶    | 上方关键位 | 手动（区间上沿/前高）         |

| D | 下方区间底    | 下方关键位 | 手动（区间下沿/前低）         |

| E | σ        | 波动范围  | 手动（如 ±5000）         |

| S | OI(张数)   | 未平仓量  | 手动/导入               |

| G | Basis偏差  | -1\~+1 | 若学生不会算，可先手填（后续可给公式） |

| H | Volume偏差 | -1\~+1 | 同上                  |



2）OI-NV 模块



| 列 | 字段      | 含义        | 公式（第2行）                                                                  |

| - | ------- | --------- | ------------------------------------------------------------------------ |

| T | OI\_NV   | 名义OI=资金口径 | `=S2\*B2`                                                                 |

| U | ΔOI\_NV% | 名义OI变化率   | `=IFERROR(T2/T1-1,"")`                                                   |

| V | μ20\_OI  | 20期均值     | `=IF(COUNT(U$2:U2)<20,"",AVERAGE(INDEX(U:U,ROW()-19):INDEX(U:U,ROW())))` |

| W | σ20\_OI  | 20期标准差    | `=IF(COUNT(U$2:U2)<20,"",STDEV.S(INDEX(U:U,ROW()-19):INDEX(U:U,ROW())))` |

| X | z\_OI    | 标准化强度     | `=IF(V2="","",(U2-V2)/MAX(W2,1E-9))`                                     |

| Y | dir\_OI  | 方向符号      | `=IF(V2="","",SIGN((B2-B1)\*U2))`                                         |

| F | OI偏差    | 输出到主模型    | `=IF(V2="","",Y2\*(MAX(-3,MIN(3,X2))/3))`                                 |



3）Bias 与安全阀（自动）



| 列 | 字段       | 含义      | 公式                                      |

| - | -------- | ------- | --------------------------------------- |

| I | Bias\_raw | 三项平均    | `=IF(COUNT(F2:H2)<3,"",AVERAGE(F2:H2))` |

| J | Bias     | 限幅后Bias | `=IF(I2="","",MAX(-0.95,MIN(0.95,I2)))` |



4）关键距离 Δup/Δdown（自动，防呆）



| 列 | 字段    | 含义    | 公式                                     |

| - | ----- | ----- | -------------------------------------- |

| K | Δup   | 到上方距离 | `=IF(OR(B2="",C2=""),"",MAX(0,C2-B2))` |

| L | Δdown | 到下方距离 | `=IF(OR(B2="",D2=""),"",MAX(0,B2-D2))` |



5）阻力 eps + 上下阻力（自动）



| 列 | 字段      | 含义   | 公式                                                |

| - | ------- | ---- | ------------------------------------------------- |

| M | eps     | 阻力下限 | `=IF(E2="","",MAX(1,0.1\*E2))`                     |

| N | UpRes   | 上方阻力 | `=IF(OR(K2="",J2="",M2=""),"",MAX(M2,K2\*(1-J2)))` |

| O | DownRes | 下方阻力 | `=IF(OR(L2="",J2="",M2=""),"",MAX(M2,L2\*(1+J2)))` |



6）方向概率



| 列 | 字段     | 含义   | 公式                                   |

| - | ------ | ---- | ------------------------------------ |

| P | P\_up   | 上行概率 | `=IF(OR(N2="",O2=""),"",O2/(N2+O2))` |

| Q | P\_down | 下行概率 | `=IF(P2="","",1-P2)`                 |



1）异常值指标选择：

OI 变化异常：用 |z\_OI|（X列）

成交活跃异常：如果你暂时没有 volume 原始值，就先用填写的 Volume偏差 的异常（H列）作为简版

（后续你可以升级为“成交额/成交量的20期z值”）



2）异常等级定义:

NORMAL：正常

ALERT：警戒（常用：|z|≥2）

EXTREME：极端（常用：|z|≥3）



| 列  | 字段         | 含义     | 公式                                                                       |

| -- | ---------- | ------ | ------------------------------------------------------------------------ |

| AA | outlier\_OI | OI异常等级 | `=IF(X2="","",IF(ABS(X2)>=3,"EXTREME",IF(ABS(X2)>=2,"ALERT","NORMAL")))` |



可选：对 H（Volume偏差）也做个简化异常（因为是 -1\~+1）：

| 列  | 字段          | 含义         | 公式                                                                             |

| -- | ----------- | ---------- | ------------------------------------------------------------------------------ |

| AB | outlier\_Vol | 成交活跃异常（简版） | `=IF(H2="","",IF(ABS(H2)>=0.85,"EXTREME",IF(ABS(H2)>=0.65,"ALERT","NORMAL")))` |



3）合成总异常等级

| 列  | 字段           | 含义    | 公式                                                                                                                         |

| -- | ------------ | ----- | -------------------------------------------------------------------------------------------------------------------------- |

| AC | outlier\_flag | 总异常等级 | `=IF(OR(AA2="",AB2=""),"",IF(OR(AA2="EXTREME",AB2="EXTREME"),"EXTREME",IF(OR(AA2="ALERT",AB2="ALERT"),"ALERT","NORMAL")))` |



**如果你觉得 AB 不想做（因为 H 是手填偏差），也可以只用 AA，然后 AC=AA。**

**出现 EXTREME 时不下方向结论，因为极端更容易扫盘/反向清算。

Step 1：先画结构关键位**

**C：区间上沿/前高（阻力）**

**D：区间下沿/前低（支撑）**

**不允许随便填。**



**Step 2：数据必须有来源**

**OI（张数）：导入**

**Basis偏差、Volume偏差：要么手填有依据，要么后续用你给的规则计算**

**σ：用当天/当周典型波动，不要拍脑袋**



**Step 3：只把模型当“方向倾向”**

**NORMAL 且 P>阈值：才叫“可执行倾向”**

**ALERT：更谨慎（阈值更高）**

**EXTREME：只做风险提示，不做方向**



**OI-NV：张数×价格 → Δ% → 20期均值/标准差 → z → dir → OI偏差(-1\~+1)**

**Bias：三项平均 + 限幅**

**Δup/Δdown：明确计算并防负值**

**eps：阻力下限（防除0/爆概率）**

**UpRes/DownRes：带eps的阻力**

**P\_up/P\_down：稳定公式**

**Outlier：OI z 异常 +（可选）Volume偏差异常**

**动态阈值：NORMAL/ALERT/EXTREME**

**结论：防呆（关键位/数据不足/极端不结论）**
