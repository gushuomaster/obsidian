2）OI-NV 模块[[OI-NV 模块]]

把「未平仓量 OI」转成“资金强度”，再做统计标准化，最后输出一个 **方向+强度信号（-1~+1）

| 列   | 字段             | 含义                        | 公式                          |
| --- | -------------- | ------------------------- | --------------------------- |
| T   | OI_NV（名义OI）    | **把“合约数量”转成“资金规模”**       | T = S × B                   |
| U   | ΔOI_NV%（变化率）   | **名义资金的增长速度**             | U = T2 / T1 - 1             |
| V   | μ20_OI（20期均值）  | **正常状态下的资金变化水平**          |                             |
| W   | σ20_OI（20期标准差） | **资金变化的正常波动幅度**           |                             |
| X   | z_OI（核心）       | **当前变化有多“异常”**(z-score)   | X = (当前变化 - 均值) / 标准差       |
| Y   | dir_OI（方向）     | **判断是谁在主导（多 / 空 / 被动清算）** | SIGN((价格变化) × (OI变化))       |
| F   | OI偏差（最终输出）     | **一个带方向的强度信号**            | F = Y × (clip(X, -3~3) / 3) |

**3）Bias 与安全阀（自动）**[[Bias和方向]]


| **列** | **字段**   | **含义**        | **公式**                       |
| ----- | -------- | ------------- | ---------------------------- |
| I     | Bias_raw | 三种市场维度的“共识方向” | I = AVERAGE(F, G, H)         |
| J     | Bias（限幅） | 限幅后Bias       | J = MAX(-0.95, MIN(0.95, I)) |

**4）关键距离 Δup/Δdown**


| **列** | **字段** | **含义** | **公式**                       |
| ----- | ------ | ------ | ---------------------------- |
| K     | Δup    | 到上方距离  | Δup = max(0, 上方关键位 - 当前价格)   |
| L     | Δdown  | 到下方距离  | Δdown = max(0, 当前价格 - 下方关键位) |


**5）阻力 eps + 上下阻力（自动）**


| **列** | **字段**    | **含义**    | **公式**                                 |
| ----- | --------- | --------- | -------------------------------------- |
| M     | eps（阻力下限） | 给一个“最小阻力” | eps = max(1, 0.1 × σ)                  |
| N     | UpRes     | 上方阻力      | UpRes = max(eps, Δup × (1 - Bias))     |
| O     | DownRes   | 下方阻力      | DownRes = max(eps, Δdown × (1 + Bias)) |

**6）方向概率（稳定直观版，自动）**



| **列** | **字段** | **含义** | **公式**                             |
| ----- | ------ | ------ | ---------------------------------- |
| P     | P_up   | 上行概率   | P_up = DownRes / (UpRes + DownRes) |
| Q     | P_down | 下行概率   | P_down = 1 - P_up                  |
